## Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/todo-app.git
cd todo-app

```

2. Install dependencies

```bash
npm install
# or
yarn install

```

3. Run the app

```bash
npx expo start

```

Scan the QR code with the Expo Go app on your device or run in an emulator
